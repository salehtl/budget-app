# Budget App

Personal finance PWA (YNAB-like). Local-first, no server dependency.

## Stack
- Runtime: Bun
- Framework: React 19 + TanStack Router (SPA)
- Build: Vite 6
- Styling: Tailwind CSS v4
- Charts: Recharts
- Database: wa-sqlite (WASM) + OPFS, IndexedDB fallback
- PWA: vite-plugin-pwa
- Validation: Zod
- Language: TypeScript

## Commands
- `bun run dev` - Start dev server
- `bun run build` - Production build
- `bun run preview` - Preview production build
